# Simple Interest Calculator

A simple bash script that calculates simple interest given principal, annual interest rate, and time period in years.

## Project Details
* **Project Name:** Simple Interest Calculator Script Project
* **Repository Name:** github-final-project
* **Course:** Introduction to Git and GitHub (IBM)
* **Author:** Đỗ Mạnh Chung

## Input Fields
1. `p`: Principal amount
2. `r`: Annual rate of interest
3. `t`: Time period in years

## Output
* `simple interest`: Calculated using the formula `p * t * r / 100`
